# Mitjul_cafe_read_helper
네이버 카페에서 봤던 게시물을 게시판에 표시하기 위해 제작하였습니다.
( 네이버 카페 일해라... )





**Ver 1.1**
코드 최적화 진행했습니다!
이제 더 적은 리로스로 동작합니다~









**#안내사항**
1. 해당 스크립트는 스트리머 밑줄 펜카페에서 적용되는 스크립트 입니다.

2. 해당 확장 프로그램을 활성화 이후 진입한 게시글에 대해서만 동작합니다.

3. 해당 스크립트 사용자에게만 표시됩니다.

4. 단 카페 홈화면 에서는 적용되지 않습니다.

5. 초기화가 필요할 경우 확장 프로그램을 재설치 하시면 됩니다.

6. 적용 방법은 크롬 확장 프로그램> 개발자도구 활성화 > 확장앱 설치 (나머지는 인터넷을 찾아서 해보시면 됩니다.)

주의사항
해당 파일이 개별로 다운로드 되는 경우 하나의 파일에 다 넣고 6번 진행하시면 됩니다.



**#알려진 이슈사항**
1. 게시글 하단에서 다음글로 진입할 경우 적용되지 않습니다.
2. 게시글 진입 후 목록으로 나가지 않고 종료시킬 경우 적용되지 않습니다.
